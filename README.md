**ATM_Simulator**



This is my 2nd year project. This project is a simple simulation of an ATM Machine. It is developed using Java, Swing, AWT, Mysql.


**It has following basic features:**
1. Login/SignUp Users
2. Deposit Money
3. Withdraw Money
4. Fast Cash
5. Mini Statement
6. Pin Change
7. Balance Enquiry
